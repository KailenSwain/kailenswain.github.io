Momentic AI test builder

Overview

Momentic enables developers to deploy AI agents to test web apps end-to-end. No code required - just describe user flows using natural language. This is a developer tool that uses AI to efficiently interact with and test an application like a real world user would. The Existing product space

Application testing is most commonly conducted using test scripts, which are unreliable and a burden to maintain as an application grows. Buy implementing a language learning model, Momentic is able to reactively adapt to the application and only surface actual issues.

Process

I am currently working closely with the co-founders to design the core user experience from the ground up. It began with initial exploration sketches that were then built out into a full Figma component library.

The test builder is the core flow of the Momentic user experience. A user creates and runs an automated QA test on their product without any coding or access to a codebase.

Test builder steps

The test builder experience revolves around creating, ordering, and editing building blocks. The blocks consist of a combination of AI steps, AI assertions, modules, preset actions, and other nodes to allow a user to create a wide variety of tests.
